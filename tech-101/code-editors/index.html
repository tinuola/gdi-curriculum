Why use an editor?

We need a place to write our code in a way that's readable for human eyes. A code editor will help us do that.

Why should I choose VS Code?

Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.

Visual Studio Code vs Visual Studio

Visual Studio is NOT the same. Visual Studio is an Integrated Development Environment (IDE)--meaning you can write and test your code at the same time. It's a great tool but it runs slow.
