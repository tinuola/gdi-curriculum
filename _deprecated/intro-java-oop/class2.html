Switch Statements

A switch statement evaluates the value of an integer or String and executes all the statements following the matching "case" block.

In other words, it evaluates the conditional once and starts execution
from there until it is told to stop or hits the end of the block.

Switch Statement Example


	switch (holiday) {
	   case "new years":
	      System.out.println("Happy New Year!");
	      break;
	   case "fourth of july":
	      System.out.println("Happy Independence Day");
	      break;
	   default:
	      System.out.println("Happy Holidays!");
	}
switch(numberPassengers){
case 1: System.out.println("I love driving by myself");
case 2: System.out.println("They always call shotgun");
case 3: System.out.println("third wheel");
case 4: System.out.println("Full house!");
default: System.out.println("over the top");
}

Expected output:
they always call shotgun
third wheel
Fullhouse!
Over the Top

Loops

With loops, we can write code once
and have it executed multiple times in a row.

For Loop

In cases where we know how many times we wish to execute the loop, we can use a for loop.

The for loop structure calls for an initialized variable,
the a termination condition, and the increment.


	for (initial variable; termination condition; increment) { }
for (int mySpeed = 0; mySpeed < 55; mySpeed += 10){
System.out.println("I'm going " + mySpeed + " mph!");
}

While Loop

At other times, we want to execute our code
for an unknown number of iterations.

It simply executes while some condition is true
regardless of how many iterations it takes.


	while (condition) {
	   ...
	}
isFull = numberPassengers == maxNumberPassengers;
while(hasSpace){
numberPassengers++;
System.out.println("I'm Carrying: "+numberPassengers);
hasSpace = numberPassengers < maxNumberPassengers;
}

Output:
I'm Carrying: 3
I'm Carrying: 4
I'm Carrying: 5

Code It!

While hasSpace is true, add a passenger
then update the value of hasSpace

The Code


	isFull = numberPassengers == maxNumberPassengers;
	while (!isFull) {
	   // We'll add a passenger
	   numberPassengers ++;

	   // We'll notifying people how many passengers we have
	   System.out.println("I'm carrying: " + numberPassengers);

	   // We'll update whether or not we're full
	   isFull = numberPassengers == maxNumberPassengers;
	}

Wrapper Classes

For each of the primitive data types we've seen,
there is a wrapper class.

A wrapper class wraps the primitive in an object.

They have the same names as the primitive types.
They're full words and start with a capital letter: Integer, Double, Boolean

Wrapper Classes

Reasons to use a number object vs. a primitive:

  1. A method expects an object
  2. Using a constant defined by the Number class
  3. Use of the Number class methods (converting to/from types or Strings and between number systems)
