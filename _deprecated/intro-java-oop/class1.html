Creating a New Java Project

  1. In Eclipse, create a new folder for your workspace, and click OK.
  2. Create a new project by selecting File > New > Project..., then select Java Project.
  3. Enter Intro to Java (or whatever you want) as your project name
  4. Choose JavaSE-1.8 as your JRE.
  5. Hit finish. Switch to the Java perspective if prompted.

Adding Our First Class

  1. Expand your project.
  2. Right click on your src folder.
  3. New > Class
  4. Enter 'HelloWorld' as your class name (no spaces!)
  5. Click finish.

Variables

What are variables?

Java is a statically typed language.

You must declare all variables before you use them.
The declaration must include the variable type and the variable name.

You can't change the type of a variable
after you declare it.

Statically Typed Variables Illustrated

You can think of variables as little boxes that hold a value.

In strongly typed langauges, variables only hold the "kind of value" they were declared to:

There are 8 total, but these are three we care about at the moment

Precedence

Just like in math class where you multiply and divide before you add and subtract, all our operators have an order of operations.

Check out the Official Documentation.

Precedence

Our previous example: 

  positiveNumbers = x > 0 && y > 0;

According to the precedence chart, evaluates:

x > 0, y > 0 relational
&& logical and
= assignment

Expressions

An expression is a piece of code that evaluates to a value.

Example:


		numberPassengers < maxNumberPassengers

This expression could be true or false depending on the values of the variables, but it is not a complete statement.

There is no semi-colon and the expression on the previous page returns a value but nothing happens.

Statement

A statement is a complete unit of execution. It's a sentence in the Java Language. An instruction that can be carried out from start to end.

Let's take a look at one of the statements we've written:


    boolean dangerToSelfOnly = numberPassengers == 1 && currentSpeed > 55;
